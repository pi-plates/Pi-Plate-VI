Overview: A very simple VI for communicating to Pi-Plates through a serial port.
Requirements: NI-VISA Driver and a Pi-Plates BRIDGEplate
Instructions:
1. Specify the BRIDGEplate COM port in the control below.
2. Type a single command in the window labeled Pi-Plate Command
2. Run the VI.
3. After the command is processed, you will see a reply from the addressed Pi-Plate in the Response window.
